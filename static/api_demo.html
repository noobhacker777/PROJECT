üéØ Product Detection API

Detect products in images using trained HOLO11 model

üìç API Endpoint

POST /api/detect

Upload an image to detect products and get count

URL: POST http://127.0.0.1:5002/api/detect

Parameters

  • image (required): Image file (JPEG, PNG, etc.)

Usage Examples

cURL:
curl -F "image=@image.jpg" \ http://127.0.0.1:5002/api/detect
Python:
import requests with open('image.jpg', 'rb') as f: resp = requests.post( 'http://127.0.0.1:5002/api/detect', files={'image': f} ) data = resp.json() print(f"Products: {data['product_count']}")
JavaScript:
const formData = new FormData(); formData.append('image', imageFile); fetch('http://127.0.0.1:5002/api/detect', { method: 'POST', body: formData }) .then(r => r.json()) .then(data => { console.log(`Found: ${data.product_count} products`); console.log(data.detections); });

üì¶ Response Format

Success Response

‚úì HTTP 200 - Detection successful
{
  "success": true,
  "product_count": 2,
  "detections": [
    {
      "id": 0,
      "confidence": 0.95,
      "box": [100.5, 200.3, 50.2, 75.1]
    },
    {
      "id": 1,
      "confidence": 0.87,
      "box": [300.2, 150.4, 55.0, 80.0]
    }
  ],
  "image_size": [2268, 4032],
  "message": "Detected 2 product(s)"
}

Response Fields

  • success: Detection succeeded (boolean)
  • product_count: Number of products found (integer)
  • detections: Array of detected products
  • image_size: [width, height] of input image
  • message: Human-readable status

Detection Object

  • id: Detection index (0-based)
  • confidence: Confidence score (0.0 - 1.0)
  • box: [x, y, width, height] in pixels

Error Response

‚úó Various error codes
{
  "success": false,
  "product_count": 0,
  "detections": [],
  "error": "Model not found. Please train first."
}
